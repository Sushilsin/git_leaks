📁 How to define git repository:
• Provide the absolute path in your prompt: /Users/username/code/my-project
• For Docker: Mount repos with -v /path/to/repos:/repos:ro then use /repos/my-project
• The path must point to a directory containing a .git folder

Example prompts:
• "Scan the repository at /Users/B0024335/code/my-project"
• "Check this code for secrets: [paste code]"
• "List all active scans"
